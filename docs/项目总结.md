## 使用说明

### 1. 环境要求

- Node.js 18+
- Python 3.8+
- npm 或 yarn

### 2. 安装依赖

```bash
# 前端
npm install

# 后端
cd backend
pip install -r requirements.txt
```

### 3. 启动项目

```bash
# 使用启动脚本 (Windows)
start.bat

# 或手动启动
npm start                    # 前端
cd backend && python app.py  # 后端
```

### 4. 访问应用

- **前端开发服务器**: http://localhost:8081
- **后端 API 服务**: http://localhost:5000
- **API 文档**: http://localhost:5000
