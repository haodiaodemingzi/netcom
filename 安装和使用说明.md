# 漫画阅读器 - 安装和使用说明

## 📦 安装步骤

### 1. 安装依赖

```bash
cd c:\coding\netcom
npm install --legacy-peer-deps
```

**注意**: 使用 `--legacy-peer-deps` 标志来解决依赖冲突

### 2. 启动APP

```bash
npm start
```

按照提示选择:
- 按 `a` 在Android设备/模拟器上运行
- 按 `i` 在iOS模拟器上运行
- 扫描二维码在Expo Go中运行

## 🌐 代理配置 (可选)

### 何时需要代理?
- ⚠️ 某些漫画网站被墙时
- ⚠️ 网络访问受限时
- ✅ 国漫8可直接访问，无需代理

### 配置步骤

1. **打开设置页面**
   - 点击底部导航栏的"设置"

2. **启用代理**
   - 打开"启用代理"开关

3. **选择代理类型**
   - HTTP (最常用)
   - HTTPS
   - SOCKS5

4. **输入代理信息**
   ```
   代理地址: 127.0.0.1
   代理端口: 7890
   ```

5. **保存设置**
   - 点击"保存设置"按钮

### 常见代理软件

| 软件 | 默认端口 | 类型 |
|------|---------|------|
| Clash | 7890 | HTTP |
| V2Ray | 10808 | SOCKS5 |
| SSR | 1080 | SOCKS5 |

## 🔧 常见问题

### Q1: 无法加载漫画?
**A**: 
1. 检查网络连接
2. 尝试切换数据源
3. 如果网站被墙，配置代理

### Q2: 图片加载失败?
**A**:
1. 检查网络
2. 重新进入章节
3. 清除APP缓存重试

### Q3: 下载的漫画在哪里?
**A**:
- 自动保存在APP内部存储
- 可在阅读器中离线查看
- 部分漫画会保存到相册

### Q4: 如何切换数据源?
**A**:
1. 点击首页右上角数据源按钮
2. 选择想要的数据源
3. 等待漫画列表刷新

### Q5: 代理不生效?
**A**:
1. 确认代理软件正在运行
2. 检查代理地址和端口正确
3. 尝试不同的代理类型
4. 重启APP
