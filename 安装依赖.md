# 依赖安装指南

## Python 依赖安装

### 方式一：使用豆瓣镜像（推荐，速度快）

```bash
cd backend
py -3 -m pip install -i https://pypi.douban.com/simple -r requirements.txt
```

### 方式二：使用批处理脚本

```bash
cd backend
install_deps.bat
```

### 方式三：配置永久镜像源

创建或编辑 `C:\Users\你的用户名\pip\pip.ini`:

```ini
[global]
index-url = https://pypi.douban.com/simple
trusted-host = pypi.douban.com
```

然后直接运行：
```bash
py -3 -m pip install -r requirements.txt
```

## JavaScript/Node.js 依赖安装

### 使用淘宝镜像（推荐）

```bash
npm install --registry=https://registry.npmmirror.com
```

### 或者配置永久镜像

```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### 恢复官方源

```bash
npm config set registry https://registry.npmjs.org
```

## 验证安装

### Python依赖
```bash
cd backend
py -3 -c "import flask; import requests; import bs4; print('所有依赖安装成功！')"
```

### Node.js依赖
```bash
npm list --depth=0
```

## 常见问题

### Q: pip安装超时
A: 使用国内镜像源，如豆瓣、清华等

### Q: npm安装慢
A: 使用淘宝镜像 `--registry=https://registry.npmmirror.com`

### Q: 权限错误
A: 使用管理员权限运行命令提示符

### Q: Python版本问题
A: 确保使用 `py -3` 而不是 `python`（如果系统默认是Python 2）
