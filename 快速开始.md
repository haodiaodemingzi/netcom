# 快速开始指南

## 📦 安装依赖

### 前端依赖

```bash
npm install
```

### 后端依赖

```bash
cd backend
pip install -r requirements.txt
cd ..
```

## 🚀 启动项目

### 方式一: 使用启动脚本 (推荐 - Windows)

双击运行 `start.bat` 文件,会自动启动前端和后端服务。

### 方式二: 手动启动

**终端 1 - 启动前端:**
```bash
npm start
```

**终端 2 - 启动后端:**
```bash
cd backend
python app.py
```

## 📱 运行应用

启动前端服务后,会显示二维码和选项:

- **按 `w`**: 在浏览器中打开 (Web)
- **按 `a`**: 在 Android 模拟器中打开
- **按 `i`**: 在 iOS 模拟器中打开
- **扫描二维码**: 在手机 Expo Go 应用中打开

## 🌐 访问地址

- **前端**: http://localhost:8081
- **后端 API**: http://localhost:5000

## 🔧 常见问题

### Q: 端口被占用怎么办?

**前端端口冲突:**
- Expo 会自动使用其他可用端口

**后端端口冲突:**
- 修改 `backend/.env` 中的 `PORT` 配置

### Q: 依赖安装失败?

**前端:**
```bash
npm cache clean --force
npm install
```

**后端:**
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### Q: 如何在真机上运行?

1. 安装 Expo Go 应用 (iOS/Android)
2. 确保手机和电脑在同一网络
3. 扫描终端显示的二维码

### Q: 后端 API 连接失败?

检查 `utils/constants.js` 中的 `API_BASE_URL` 配置:
- 模拟器: `http://localhost:5000/api`
- 真机: `http://你的电脑IP:5000/api`
